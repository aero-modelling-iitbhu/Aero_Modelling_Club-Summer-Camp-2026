# Week 0: Installation Week ⚙️

Welcome to **Week 0: Installation Week**! Before we take off into aerodynamics, simulation, and autonomous drone flight, we need to prepare our flight controllers and ground stations. 

This week is dedicated to setting up your systems so that all simulation software and engineering tools run seamlessly.

---

## 🛠️ Essential Software Suite

Here are the step-by-step guides and resources to install all the tools required throughout the camp.

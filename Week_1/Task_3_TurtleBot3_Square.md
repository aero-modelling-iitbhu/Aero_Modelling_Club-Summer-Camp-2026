# 🐢 Task 3: TurtleBot3 Navigation Challenge

## **Introduction**

Welcome to the TurtleBot3 Navigation Challenge! This task is designed to help you practice robot programming fundamentals using ROS 2 and the TurtleBot3 simulation platform. After successfully installing TurtleBot3 with Gazebo, it's time to apply your knowledge to a practical motion control challenge.

In this task, you will create a Python script that makes the TurtleBot3 robot move in a square pattern within a simulated environment. This exercise will help you understand ROS topics, message publishing, and basic robot motion planning.

---

## **Task Description**

Your challenge is to program the TurtleBot3 to navigate in a perfect square pattern within the Gazebo simulation environment. 

The robot should:
1. Start from its initial position.
2. Move forward along a straight line for a sharp fixed distance.
3. Make a sharp turn of exactly 90 degrees.
4. Repeat steps 2 and 3 three more times to complete the square.
5. Return to its approximate starting position.

> **Note:** Don’t use AI tools! The goal is to help you learn and understand the core basics of `/cmd_vel` velocity commands on your own.
